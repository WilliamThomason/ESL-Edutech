# Eloquenta Main

The Eloquenta platform shell — landing pages, authentication, dashboard, teacher studio, and curriculum overview.

## What's In This Repo

- `landing.html` — Main Eloquenta landing page
- `login.html` — Login/authentication
- `dashboard.html` — Student dashboard
- `teacher-studio.html` — Teacher tools and class management
- `eloquent-curriculum.html` — Curriculum overview (links to knowledge base)
- `speaking-catalog.html` — Speaking practice slideshow catalog
- `cal-com-bio.html` — Booking page
- `eloquent-branding.css` — Shared Eloquenta styles

## Deployment

This is a static HTML/CSS/JS site. Deploy to any web server or static host:

```bash
# GitHub Pages: push to gh-pages branch
# Netlify: drag and drop this folder
# Vercel: vercel deploy
# Any server: scp -r . user@server:/var/www/eloquenta-main
```
