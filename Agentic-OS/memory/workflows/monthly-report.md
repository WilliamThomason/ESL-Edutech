# Workflow: Monthly SEO Report

## Trigger
First day of each month (or client request)

## Steps
1. **Analyst** — Pull ranking data for all tracked keywords
2. **Analyst** — Pull traffic data (GA4 / Search Console)
3. **Analyst** — Calculate month-over-month changes
4. **Writer** — Draft narrative summary of performance
5. **Analyst** — Generate charts and visualizations
6. **Reviewer** — Quality check report accuracy
7. **COO** — Approve and send to client

## Token Budget
- Analyst: 60K
- Writer: 20K
- Reviewer: 10K
- COO: 5K
- **Total: ~95K per report**

## Output
- PDF report
- Client email with key insights
- Updated campaign tracker
